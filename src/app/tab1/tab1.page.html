Guia de Uso

Este aplicativo permite que você utilize a câmera para tirar fotos e armazená-las em uma galeria local.

Siga as instruções para aproveitar ao máximo o aplicativo.

1. Para acessar a câmera, toque no ícone de fotos localizado na parte inferior da tela.

2. Dentro da área de fotos, clique no botão “Câmera” para abrir a câmera do dispositivo.

3. Tire a foto normalmente, aceitando ou repetindo conforme necessário.

4. Após capturada, a imagem é salva automaticamente no aplicativo.
